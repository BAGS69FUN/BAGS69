## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/bags69-launchpad.git`
3. Install dependencies: `pnpm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
# Start development server
pnpm dev

# Run type checking
pnpm typecheck

# Run linting
pnpm lint
```

## Environment Setup

Copy `.env.example` to `.env.local` and fill in your values:

```bash
cp .env.example .env.local
```

## Code Style

- Use TypeScript for all new code
- Follow existing code patterns
- Add comments for complex logic
- Use meaningful variable names

## Pull Request Process

1. Update documentation if needed
2. Test your changes thoroughly
3. Create a PR with a clear description
4. Wait for review

## Security

If you discover a security vulnerability, please email security@bags69.fun instead of creating a public issue.
